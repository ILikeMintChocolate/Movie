### 다운로드 파일 실행방법

```
npm install
npm run dev
```



### 주의사항

* 캐시 문제로 인해 시크릿탭을 새로 열어서 시연 해보시는 것을 추천드립니다.
* 입력된 데이터들은 데이터베이스에 저장되기 때문에 '같은 특수기호 사용은 삼가주시기 바랍니다.
* 관리자 페이지에 접속 하기 위해 이메일:hs@movie.com, 비밀번호:1234 로 로그인 하시는 것을 권장해 드립니다.
